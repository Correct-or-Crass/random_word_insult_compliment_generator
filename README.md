## This API returns a random insult or compliment

## Root directory
'https://shielded-headland-00998.herokuapp.com'

## API Endpoints
- 'GET  /api/v1/insults'

- Successful response:
```
{
  "data": {
      "id": "7",
      "type": "insult",
      "attributes": {
            "insult_phrase": "You have miles to go before you reach mediocre."
            }
      }
}
```
